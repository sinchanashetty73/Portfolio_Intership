# Personal Portfolio Website

A modern and responsive personal portfolio website developed using HTML and CSS. This project showcases personal information, technical skills, projects, and contact details in a clean and professional layout.

## Features

* Responsive design for desktop, tablet, and mobile devices
* Fixed navigation bar with smooth scrolling
* Hero section with profile image and introduction
* About section with personal and educational information
* Technical Skills section with animated skill cards
* Projects section displaying project details and images
* Contact section with contact information and contact form
* CSS animations, hover effects, and modern UI styling
* Organized and easy-to-maintain project structure

## Technologies Used

* HTML5
* CSS3
* Flexbox
* CSS Grid
* Media Queries
* Font Awesome Icons

## Project Sections

* Home
* About
* Skills
* Projects
* Contact

## Purpose

This portfolio website was created to showcase my skills, projects, and professional profile as an AIML Engineering student and aspiring Full Stack Developer.
